# Angimen Holding — Web

Property Management · Construction · Renovation · Maintenance · Marbella.

## Estructura
- `index.html` — selector ES de las 3 propuestas
- `en/index.html` — selector EN
- `a/`, `b/`, `c/` — 3 propuestas ES (Marbella Authority, Wellness Estate, Editorial Architecture)
- `en/a/`, `en/b/`, `en/c/` — 3 propuestas EN
- `assets/` — imágenes + JS compartido
- `404.html` — página de error

## Stack
HTML5 estático + CSS3 + JS vanilla. Cormorant Garamond + Inter (Google Fonts). Mobile first.
