# Angimen Holding — Web v1

Web premium para Angimen Holding (Property Management · Construction · Maintenance, Marbella).

## Estructura
- `index.html` — versión ES (raíz)
- `en/index.html` — versión EN
- `assets/css/main.css` — estilos
- `assets/js/main.js` — JS (nav, scroll, filtros, form)
- `404.html` — página de error

## Stack
- HTML5 estático + CSS3 + JS vanilla (sin framework)
- Cormorant Garamond + Inter (Google Fonts)
- Mobile first, breakpoints 720/1024/1280px
- Compatible con GitHub Pages, Cloudflare Pages, Netlify, Vercel

## Pendiente para deploy público
- Datos cliente reales (teléfono, email, razón social, CIF)
- Dominio
- Fotos reales con autorización formal Metro Grupo / restaurantes
- Aviso legal + privacidad + cookies
- Gate verify_web.py + Lighthouse + axe + Mozilla Observatory
- Quitar `noindex,nofollow` de meta robots
- Quitar `Disallow: /` de robots.txt
